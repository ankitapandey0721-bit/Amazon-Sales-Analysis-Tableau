# Amazon Annual Sales Analysis (2026) 📊

## Project Overview
Is project mein maine Amazon ke sales data ka analysis kiya hai taaki business growth aur customer trends ko samjha ja sake. Maine raw data ko Excel mein clean kiya aur phir Tableau ka use karke ek interactive dashboard banaya.

## Tools Used
* **Data Visualization:** Tableau
* **Data Cleaning:** Microsoft Excel (Pivot Tables, VLOOKUP)
* **Analysis:** Sales Trends, Regional Performance

## Key Insights (Jo maine find kiya)
* **Top State:** Maharashtra ne sabse zyada revenue ($13.34M) generate kiya.
* **Top Category:** Electronics aur Clothing mein sabse zyada demand dekhi gayi.
* **Growth:** Year-over-year sales mein steady growth observe hui.
